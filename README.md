# ⚙️ Akroud Parfum - Backend (RESTful API)

A robust and secure API that manages the catalog and orders for the Akroud Parfum e-commerce platform. Built with Laravel to ensure high performance and data security.

## 🚀 Features
* **Order Management:** Strict validation and processing of client data (Name, Phone, City, Payment Method, etc.).
* **API Endpoints:** Clean route separation (`/api/perfumes` for the catalog, `/api/commandes` for checkout) to prevent conflicts.
* **Relational Database:** Secure storage of customer information and order history.

## 🛠️ Technologies Used
* **Framework:** Laravel (PHP)
* **Database:** MySQL
* **Architecture:** MVC / REST API
